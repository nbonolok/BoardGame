Game Interface

Mhele game interface consists of three sections.which are as follows

TaskBar:this is were the user actives game commands by clicking on different menuItems.

Game Board:this is the play area for mhele game,the player has to click on the board nodes or game caps to perform a move.

Deck Panel:This is used by the player to inform him/ her about the status of the game.

Game Menu

New Game:Start a new Mhele game.

Load Game: Clicking on this menu option will load or restore a game that was saved during the last session of game play.

Save Game:Clicking on this menu option will save the current game so that it can be played on a later stage.

Quit:This menuitem will close the current game.

Exit:By Clicking on this option the game application will be terminated.

 

Option Menu

Game Board:changing the Mhele Board or game Themes.

Player: This option is used for selecting game modes which are as follows

  1. Player Name:Player can change a default name to his or her own name.
  2. Two Players: When two Human players face each other.
    One Player: This is the default game mode which lets one player to play against a computer.

Cows : This option is used by a player for changing the number of Cows used in a game which are 6,9 and 12.Cows option is also used for changing the look and fell of the Cows.

Sound:This is used for switching ON or OFF the game sound Effects

Langauge:A player has an option of choosing between English and Setswana Langauges.

 

Help Menu

Game Manual: This will open an electronic game manual.

About Mhele: Creadits and Developer information.

This panel is where a record of current game status is viewed.

Player Name

Name of a player either Human or computer.

Cows that a player has: This are tokens or cows that still belong to a player ,not yet placed on the game board or captured.

Captured Cows from Opponent: All cows or token captured by an opponent from the game board after a three in a row set was made.

Player that has to Play: This feature helps players by keeping track of a player that has to move by circling a token that belong to that player.
