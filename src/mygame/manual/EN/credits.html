Ngwao Ke Boleng Jwa me; Ke Ipela Ka Yone (Culture: my roots; my dignity; My Pride). Mhele is a board game which has its roots in Africa. This is a traditional game that had been played for generations in Botswana and some parts of Africa.

Mhele game is designed and development by Bonolo K Nyoni as a hobby.The entire game is written in Java (J2SE).It has some known bugs as this was the first game the developer ever created.The developer never had formal training in game design.

Best Student Computer Engineering Project Design (Robotics) EXPO-2006 at Tshwane University Of Technology (SA)
