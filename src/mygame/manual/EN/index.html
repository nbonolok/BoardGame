Mhele version 0.0.1

Ngwao Ke Boleng Jwa me; Ke Ipela Ka Yone (Culture: my roots; my dignity; My Pride). Mhele is a board game which has its roots in Africa. This is a traditional game that had been played for generations in Botswana and some parts of Africa.

Mhele also known as Morabarabara represents the wealth of African people as game token used in the game are called Cows (Dikgomo) and the game Board called a Kraal. This game is played by two players each one with 12 tokens and having the main object of capturing all tokens that belongs to an opponent.Â 

NB: A player in Mhele board game is called Modisa which is a Setswana word for herdboy.
